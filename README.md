Full Stack Development Workshop Project
_______________________________________

This project was developed during the Full Stack Development Workshop (7th–8th February 2026).
The objective was to build a secure and scalable backend system using modern industry tools and technologies.

Tech Stack Used:

Node.js – Server-side runtime environment
Express.js – Backend framework for building APIs
PostgreSQL – Relational database
Drizzle ORM – Database ORM for structured queries
Supabase – Database hosting & backend services
JWT (jsonwebtoken) – Secure authentication & authorization
Postman – API testing
Ngrok – Exposing local server to the internet

Project Description:

In this workshop, we built a backend API system with authentication and database integration.
The application includes:
User Registration & Login System
Secure Authentication using JWT
Database connectivity with PostgreSQL
RESTful API creation using Express
API testing and debugging using Postman
Deployment testing using Ngrok

Key Features:

Secure user authentication
Token-based authorization
Structured database schema
CRUD operations
API endpoint testing
